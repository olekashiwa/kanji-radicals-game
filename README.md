# 📖 Kanji Radicals Game

Интерактивное веб-приложение для изучения всех 214 японских радикалов (ключей) с системой тестирования, игровыми элементами и таблицей лидеров.

## 🌟 Особенности

- **214 радикалов** — полная база ключей Канси с номерами, названиями, чтениями (онъёми/кунъёми) и примерами
- **Режимы обучения**:
  - 📚 Изучение — карточки с информацией о каждом радикале
  - 📝 Тестирование — проверка знаний с подсчётом баллов
  - ⚔️ Битва — соревнование с другими пользователями
- **Фильтрация и поиск** — быстрый поиск по номеру, символу или значению
- **Таблица лидеров** — глобальный рейтинг игроков (Firebase)
- **Экспорт в Anki** — выгрузка изученных радикалов для повторения
- **Адаптивный дизайн** — работает на ПК, планшетах и смартфонах

## 🛠️ Технологии

- HTML5 / CSS3 / JavaScript (ES6 Modules)
- Firebase Authentication — анонимный вход и авторизация
- Firestore — хранение прогресса и таблицы лидеров
- Адаптивная вёрстка (Flexbox, Grid, Media Queries)
